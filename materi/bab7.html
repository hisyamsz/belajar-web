Bab 7

CSS Frameworks Modern

Pada bab ini kamu akan mempelajari penggunaan CSS Frameworks modern seperti Bootstrap dan Tailwind CSS untuk mempercepat proses styling dan pembuatan komponen antarmuka website.

CSS Framework menyediakan kumpulan class dan komponen siap pakai sehingga pembuatan layout dan tampilan website dapat dilakukan lebih cepat dan konsisten.

  • Bootstrap ➜ berbasis komponen & grid system
  • Tailwind CSS ➜ berbasis utility class
  • Mengurangi penulisan kode CSS manual
  • Membantu standarisasi tampilan

<!-- Bootstrap via jsDelivr -->
<link rel="stylesheet"
href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">

<!-- Tailwind CDN -->
<script src="https://cdn.tailwindcss.com"></script>

Tailwind CSS menggunakan pendekatan utility-first sehingga style dapat langsung diterapkan pada elemen HTML melalui class pendukung. 


<button class="px-4 py-2 bg-blue-600 text-white rounded-lg shadow">
  Tombol Aksi
</button>

<div class="p-4 mt-3 border rounded-lg bg-gray-100">
  <h3 class="text-lg font-semibold">Card Tailwind</h3>
  <p class="text-sm text-gray-600">Contoh komponen sederhana</p>
</div>
  • Pembuatan komponen lebih cepat
  • Kode tampak ringkas & efisien
  • Sangat mendukung pengembangan UI modern

Framework membantu percepatan pengembangan, namun pemahaman CSS murni tetap diperlukan agar pengembang memiliki kendali penuh terhadap tampilan dan perilaku elemen.

  • Framework ➜ cepat & praktis
  • CSS Murni ➜ fleksibel & detail
  • Keduanya saling melengkapi

/* CSS Murni */
.button {
  background: blue;
  color: white;
  padding: 10px;
}

/* Tailwind */
class="bg-blue-600 text-white px-3 py-2"
