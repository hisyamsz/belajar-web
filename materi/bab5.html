Bab 5

Layout Dasar dengan CSS

Pada bab ini kamu akan mempelajari dasar penyusunan layout halaman web menggunakan CSS, dimulai dari konsep box model, sistem positioning, hingga pengaturan display dan visibility elemen.

Setiap elemen HTML memiliki struktur box model yang terdiri dari content, padding, border, dan margin. Konsep ini penting untuk mengatur jarak dan tata letak elemen pada halaman web.

  • Content ➜ isi elemen
  • Padding ➜ jarak di dalam elemen
  • Border ➜ garis tepi elemen
  • Margin ➜ jarak antar elemen

.card {
  width: 200px;
  padding: 12px;
  margin: 10px;
  border: 2px solid #0d6efd;
  border-radius: 10px;
}

Positioning digunakan untuk mengatur penempatan elemen pada layar. Setiap nilai posisi memiliki fungsi yang berbeda sesuai kebutuhan layout halaman web.

  • static ➜ posisi default elemen
  • relative ➜ bergerak relatif terhadap posisi awal
  • absolute ➜ diposisikan terhadap parent terdekat
  • fixed ➜ menempel pada layar (tidak ikut scroll)

.box-relative {
  position: relative;
  top: 10px;
  left: 10px;
}

.box-absolute {
  position: absolute;
  top: 20px;
  right: 20px;
}

.nav-fixed {
  position: fixed;
  top: 0;
  width: 100%;
}

Properti display mengatur bagaimana elemen ditampilkan dalam layout, sedangkan visibility menentukan apakah elemen terlihat atau disembunyikan.

  • display: block ➜ elemen menempati satu baris penuh
  • display: inline ➜ sebaris dengan elemen lain
  • display: inline-block ➜ bisa diatur ukuran
  • visibility: hidden ➜ elemen disembunyikan

.block {
  display: block;
}

.inline {
  display: inline;
}

.inline-block {
  display: inline-block;
  width: 120px;
}

.hidden {
  visibility: hidden;
}
