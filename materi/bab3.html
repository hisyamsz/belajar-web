Struktur Semantik HTML5

Pada bab ini kamu akan mempelajari penggunaan elemen semantik dalam HTML5 yang membantu menjelaskan struktur halaman web, meningkatkan keterbacaan kode, serta memudahkan optimasi SEO dan aksesibilitas.

Elemen semantik digunakan untuk menjelaskan fungsi bagian-bagian halaman sehingga struktur web menjadi lebih teratur dan mudah dipahami.

  • <header> ➜ bagian kepala halaman / judul
  • <nav> ➜ menu navigasi
  • <section> ➜ pengelompokkan konten
  • <article> ➜ konten utama / artikel mandiri
  • <footer> ➜ informasi bagian bawah halaman

<header>
  <h1>Belajar Web Development</h1>
</header>

<nav>
  <a href="#">Home</a>
  <a href="#">Materi</a>
</nav>

<section>
  <article>
    <h2>Struktur Semantik HTML5</h2>
    <p>Contoh penggunaan elemen semantik.</p>
  </article>
</section>

<footer>
  <p>© 2026 Kelompok 5</p>
</footer>

Elemen semantik membantu mesin pencari dan screen reader memahami struktur konten sehingga halaman lebih ramah SEO dan mudah diakses.

  • Meningkatkan struktur dokumen
  • Mempermudah Google membaca konten
  • Mendukung pengguna disabilitas (screen reader)
  • Kode lebih rapi dan mudah dipelihara

Berikut contoh latihan pembuatan halaman blog sederhana dengan struktur semantik HTML5. 


<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Blog Teknologi Masa Depan</title>
</head>

<body>

<header>
  <h1>TechVision5 Blog</h1>
  <p>Wawasan Teknologi Terkini untuk Indonesia</p>
</header>

<nav>
  <ul>
    <li><a href="#home">Beranda</a></li>
    <li><a href="#tutorial">Tutorial</a></li>
    <li><a href="#news">Berita</a></li>
    <li><a href="#contact">Kontak</a></li>
  </ul>
</nav>

<div class="container">

  <main>
    <article>

      <header>
        <h2>Mengenal HTML5 Semantik</h2>
        <p>Ditulis oleh <strong>Admin</strong> pada 10 Oktober 2025</p>
      </header>

      <section>
        <h3>Apa itu Semantik?</h3>
        <p>Semantik berarti kode yang menjelaskan dirinya sendiri dan
        membantu mesin pencari memahami konten.</p>
      </section>

      <section>
        <h3>Mengapa Penting?</h3>
        <p>Penting untuk SEO dan aksesibilitas. Berikut manfaatnya:</p>
        <ol>
          <li>Kode lebih rapi</li>
          <li>Ramah screen reader</li>
          <li>Ranking Google lebih baik</li>
        </ol>
      </section>

      <footer>
        <p>Tags:
          <a href="#">HTML</a>,
          <a href="#">Coding</a>
        </p>
      </footer>

    </article>
  </main>

  <aside>
    <h3>Tentang Penulis</h3>
    <img src="gambar/penulis.jpg" alt="Foto Profil Penulis" width="100">
    <p>Seorang pengembang web yang antusias berbagi ilmu.</p>
