Bab 3

Pengenalan CSS Dasar

Pada bab ini kamu akan mempelajari dasar-dasar CSS, fungsi CSS dalam mengatur tampilan website, penggunaan selector, properti gaya, serta cara mengatur warna, teks, jarak, dan layout elemen.

CSS digunakan untuk mengatur tampilan dan desain halaman web. Dengan CSS, elemen HTML dapat diberi warna, ukuran, tata letak, dan gaya visual agar website lebih rapi dan menarik.

  • CSS memisahkan konten (HTML) dan tampilan (style)
  • Memudahkan pengelolaan desain halaman web
  • Satu file CSS dapat digunakan pada banyak halaman

CSS dapat ditulis menggunakan tiga metode, yaitu inline style, internal style di dalam tag head, dan external stylesheet menggunakan file terpisah. 


/* External CSS */
<link rel="stylesheet" href="style.css">

/* Internal CSS */
<style>
  h1 { color: blue; }
</style>

/* Inline CSS */
<p style="color:red;">Teks berwarna merah</p>

Selector digunakan untuk memilih elemen yang akan diberikan style. Class digunakan untuk banyak elemen, sedangkan ID hanya untuk satu elemen khusus.

  • Tag selector : memilih berdasarkan nama elemen
  • Class selector : menggunakan tanda titik (.)
  • ID selector : menggunakan tanda pagar (#)

h1 { color: green; }

.container { padding: 20px; }

#judul-utama { font-weight: bold; }

CSS menyediakan properti untuk mengatur warna, jarak di luar elemen (margin), serta jarak di dalam elemen (padding) agar tampilan lebih terstruktur dan nyaman dilihat.

  • color = mengatur warna teks
  • background-color = memberi warna latar
  • margin = jarak luar elemen
  • padding = jarak dalam elemen

.box {
  color: white;
  background-color: #0d6efd;
  margin: 10px;
  padding: 15px;
  border-radius: 10px;
}
