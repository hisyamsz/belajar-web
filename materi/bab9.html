Bab 9

Manipulasi DOM (Document Object Model)

Pada bab ini kamu akan mempelajari bagaimana JavaScript berinteraksi dengan elemen HTML melalui DOM untuk menampilkan, mengubah, dan memproses konten secara dinamis.

DOM memungkinkan JavaScript memilih elemen tertentu pada halaman menggunakan berbagai metode pemilihan elemen.

  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()

<h2 id="judul">Belajar DOM</h2>

<script>
  const teks1 = document.getElementById("judul");
  const teks2 = document.querySelector("h2");

  console.log(teks1.innerText);
</script>

JavaScript dapat memodifikasi konten halaman secara langsung seperti mengganti teks, menambah atribut, atau mengubah style CSS.

  • innerText & innerHTML
  • setAttribute()
  • style.property

<p id="info">Status Awal</p>

<script>
  const el = document.getElementById("info");

  el.innerText = "Teks berhasil diubah";
  el.style.color = "blue";
  el.setAttribute("class", "highlight");
</script>

Event digunakan untuk menjalankan aksi tertentu ketika pengguna berinteraksi dengan elemen seperti klik tombol atau submit form.

  • onclick ➜ event klik tombol
  • oninput ➜ perubahan input
  • onsubmit ➜ pengiriman form

<button onclick="sapa()">Klik Saya</button>

<script>
  function sapa() {
    alert("Event onclick berjalan");
  }
</script>
