Bab 2

Struktur Dasar HTML5

Pada bab ini kamu akan mempelajari penyusunan struktur halaman menggunakan elemen HTML, cara penggunaan tag dan atribut, penulisan heading, paragraf, link, gambar, list, table, serta dasar pembuatan formulir dan input pada halaman web.

HTML dibangun dari elemen-elemen yang ditulis menggunakan tag. Sebagian elemen memiliki atribut yang berfungsi memberikan informasi tambahan pada suatu tag.

  • Tag ➜ penanda awal dan akhir elemen
  • Elemen ➜ terdiri dari tag pembuka, isi, dan tag penutup
  • Atribut ➜ menambahkan properti pada elemen

<p title="deskripsi paragraf">
  Ini adalah elemen paragraf dengan atribut.
</p>

HTML menyediakan berbagai elemen untuk menampilkan teks, tautan, gambar, daftar, dan tabel sebagai penyusun konten halaman.

  • Heading h1–h6 ➜ judul & subjudul
  • Paragraf ➜ elemen <p>
  • Hyperlink ➜ elemen <a>
  • Gambar ➜ elemen <img>
  • List ➜ <ul>, <ol>, <li>
  • Table ➜ <table>, <tr>, <th>, <td>

<h1>Judul Halaman</h1>
<p>Ini adalah paragraf pertama.</p>

<a href="https://example.com">Kunjungi Website</a>

Formulir digunakan untuk menerima data dari pengguna melalui elemen input seperti text field, email, password, dan tombol submit.

  • <form> ➜ membungkus input pengguna
  • <input> ➜ text, email, password, number
  • <label> ➜ penanda nama field
  • <button> ➜ mengirim data

<form>
  <label>Nama</label>
  <input type="text" placeholder="Masukkan nama">

  <label>Email</label>
  <input type="email" placeholder="Masukkan email">

  <button type="submit">Kirim</button>
</form>
