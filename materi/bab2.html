Struktur Dasar HTML5

Pada bab ini kamu akan mempelajari struktur dasar dokumen HTML5, elemen dan atribut, penggunaan heading dan paragraf, serta elemen konten seperti hyperlink, gambar, list, tabel, dan elemen semantik.

HTML tersusun dari elemen-elemen yang ditulis menggunakan tag. Setiap elemen dapat memiliki atribut untuk memberikan informasi tambahan pada suatu tag.

  • Tag berfungsi sebagai penanda struktur dokumen
  • Elemen terdiri dari tag pembuka, isi, dan tag penutup
  • Atribut digunakan untuk menambahkan properti pada elemen

Dokumen HTML memiliki struktur utama yang terdiri dari DOCTYPE, elemen html, bagian head untuk metadata, dan body sebagai tempat menampilkan konten halaman. 


<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Struktur HTML5</title>
  </head>

  <body>
    Konten halaman ditulis pada bagian body.
  </body>
</html>

Heading digunakan sebagai judul dan subjudul yang memiliki tingkat hierarki, sedangkan paragraf digunakan untuk menuliskan isi teks.

  • Heading terdiri dari h1 sampai h6
  • h1 digunakan sebagai judul utama halaman
  • Paragraf ditulis menggunakan elemen <p>

<h1>Judul Utama</h1>
<h2>Subjudul Materi</h2>

<p>Ini adalah contoh paragraf pada dokumen HTML.</p>

HTML menyediakan berbagai elemen untuk menampilkan konten seperti tautan, gambar, daftar, serta tabel sebagai penyaji data terstruktur.

  • <a> digunakan untuk membuat hyperlink
  • <img> digunakan untuk menampilkan gambar
  • <ul> dan <ol> digunakan untuk membuat daftar
  • <table> digunakan untuk menampilkan data dalam baris & kolom

<a href="https://example.com">Kunjungi Website</a>
