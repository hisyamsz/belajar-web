Membangun Website Responsif

Pada bab ini kamu mempelajari konsep Responsive Web Design (RWD), penggunaan Media Queries pada CSS, serta penerapan responsivitas menggunakan Tailwind CSS.

Responsive Web Design adalah pendekatan desain web yang membuat tampilan website menyesuaikan ukuran layar perangkat seperti smartphone, tablet, maupun desktop.

  • Konten dapat menyesuaikan ukuran layar
  • Layout fleksibel
  • Teks tetap mudah dibaca
  • Elemen tidak saling bertabrakan

Contoh Struktur Layout Fleksibel


<div class="container">
  <div class="card">Konten 1</div>
  <div class="card">Konten 2</div>
  <div class="card">Konten 3</div>
</div>


.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Media Query digunakan untuk mengubah layout berdasarkan ukuran layar device.

Breakpoint Umum

  • max-width: 1200px → Laptop
  • max-width: 992px → Tablet
  • max-width: 768px → Tablet kecil
  • max-width: 576px → Smartphone

Contoh Media Query


@media (max-width: 768px) {
  .container {
    grid-template-columns: repeat(2, 1fr);
  }
}


@media (max-width: 480px) {
  .container {
    grid-template-columns: 1fr;
  }
}

Tailwind CSS menyediakan kelas responsif bawaan menggunakan prefix breakpoint.

  • sm: Smartphone
  • md: Tablet
  • lg: Laptop
  • xl: Desktop

Contoh Grid Responsif


<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  <div class="p-4 bg-slate-200">Box 1</div>
  <div class="p-4 bg-slate-300">Box 2</div>
  <div class="p-4 bg-slate-400">Box 3</div>
</div>

Contoh Button Responsif


<button class="px-6 py-2 bg-blue-600 text-white sm:text-sm md:text-base lg:text-lg rounded-lg">
  Tombol Responsif
</button>
