CSS Flexbox dan Grid

Pada bab ini kamu akan mempelajari penyusunan layout modern menggunakan CSS Flexbox dan CSS Grid untuk membuat tampilan web yang responsif, terstruktur, dan mudah diatur.

Flexbox digunakan untuk mengatur tata letak elemen secara fleksibel dalam satu baris atau kolom. Flexbox memudahkan penyusunan layout yang dinamis dan responsif.

  • flex-direction ➜ mengatur arah baris / kolom
  • justify-content ➜ perataan secara horizontal
  • align-items ➜ perataan secara vertikal
  • gap ➜ jarak antar elemen

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 10px;
}

CSS Grid digunakan untuk membuat layout berbasis kolom dan baris. Grid sangat cocok untuk membangun struktur halaman yang kompleks namun tetap rapi dan responsif.

  • grid-template-columns ➜ jumlah & ukuran kolom
  • grid-template-rows ➜ jumlah & ukuran baris
  • gap ➜ jarak antar grid item
  • responsive layout menggunakan fr unit

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 12px;
}

Pada studi kasus ini kamu akan melihat penerapan Flexbox pada header dan hero section, serta penggunaan CSS Grid untuk menampilkan galeri project dalam layout landing page sederhana.

CSS Flexbox & Grid


/* Reset CSS Dasar */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
  font-family: sans-serif;
}

/* 1. Implementasi Flexbox pada Header */
.main-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px 50px;
  background-color: #2c3e50;
  color: white;
}

.main-header nav a {
  color: white;
  text-decoration: none;
  margin-left: 20px;
}

/* 2. Implementasi Flexbox pada Hero Section */
.hero {
  display: flex;
  align-items: center;
  padding: 100px 50px;
  background-color: #ecf0f1;
  min-height: 60vh;
}

.hero-text {
  flex: 1;
  padding-right: 50px;
}

.hero-image {
  flex: 1;
}

.kotak-gambar {
  width: 100%;
  height: 300px;
  background-color: #bdc3c7;
  display: flex;
  justify-content: center;
  align-items: center;
  border-radius: 10px;
}

/* 3. Implementasi Grid pada Galeri */
.projects {
  padding: 50px;
}

.grid-gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 30px;
  margin-top: 30px;
}

.card {
  background-color: white;
  border: 1px solid #ddd;
  height: 200px;
  padding: 20px;
  box-shadow: 0 4px 6px rgba(0,0,0,0.1);
  border-radius: 8px;
  transition: transform 0.3s ease;
}

.card:hover {
  transform: translateY(-5px);
}
