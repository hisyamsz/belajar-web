Bab 11

Membuat Website Statis Sederhana

Pada bab ini kamu akan mempraktikkan pembuatan website statis sederhana menggunakan HTML5 dan CSS3 mulai dari struktur dasar halaman, header dan navigasi, hero section, project section, hingga styling layout menggunakan Flexbox dan Grid.

Tahap awal pembuatan website dimulai dengan menyiapkan kerangka dasar file HTML dan file CSS. Pada contoh berikut digunakan Google Fonts serta CSS Variables untuk mendefinisikan warna dan font utama.

Kode HTML


<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Portofolio Profesional | Kelompok 5</title>

    <link rel="stylesheet" href="assets/css/style.css" />

    <link
      href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600&display=swap"
      rel="stylesheet"
    />
  </head>

  <body>

  </body>
</html>

Kode CSS


/* 1. CSS Variables: Menyimpan nilai warna dan font */
:root {
  --primary-color: #2563eb;   /* Biru Utama */
  --secondary-color: #1e293b; /* Biru Gelap */
  --light-bg: #f8fafc;        /* Abu-abu muda */
  --white: #ffffff;
  --font-main: 'Poppins', sans-serif;
}

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: var(--font-main);
  color: var(--secondary-color);
  background-color: var(--light-bg);
  line-height: 1.6;
}

.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 20px;
}

A. Styling Header (Flexbox)
