Bab 8

Dasar JavaScript untuk Pemula

Pada bab ini kamu akan mempelajari konsep dasar pemrograman JavaScript sebagai bahasa pemrograman utama pada pengembangan web modern.

Variabel digunakan untuk menyimpan data. JavaScript mendukung beberapa tipe data seperti number, string, boolean, array, dan object.

  • let ➜ variabel yang masih dapat diubah
  • const ➜ variabel bernilai tetap
  • operator aritmatika dan logika

<script>
  let nama = "Hisyam";
  const umur = 20;

  let total = 10 + 5;
  let aktif = true;

  console.log(nama);
  console.log(total);
</script>

Fungsi digunakan untuk membungkus logika program agar dapat dipanggil berulang kali. Scope menentukan jangkauan variabel di dalam blok kode.

  • function declaration
  • parameter & return value
  • block scope pada let & const

<script>
  function sapaUser(nama) {
    return "Halo " + nama;
  }

  let pesan = sapaUser("Mahasiswa");
  console.log(pesan);
</script>

Struktur kontrol digunakan untuk mengambil keputusan dan mengulang proses sesuai kondisi tertentu.

  • if – else ➜ percabangan kondisi
  • for ➜ perulangan dengan counter
  • while ➜ perulangan berdasarkan kondisi

<script>
  let nilai = 85;

  if (nilai >= 80) {
    console.log("Lulus");
  }

  for (let i = 1; i <= 3; i++) {
    console.log("Perulangan ke-" + i);
  }
</script>
