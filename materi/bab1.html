Bab 1

Pendahuluan Pemrograman Web

Pada bab ini kamu akan mempelajari dasar-dasar pemrograman web, perbedaan front-end dan back-end, alur kerja pengembangan website, tools yang digunakan programmer web, serta pengenalan dasar HTML.

Pemrograman web adalah proses membuat aplikasi atau halaman web yang dapat diakses melalui browser menggunakan teknologi seperti HTML, CSS, dan JavaScript.

  • Digunakan untuk membuat website & aplikasi berbasis web
  • Dapat diakses melalui internet maupun jaringan lokal
  • Berjalan di sisi client, server, atau keduanya

Dalam pengembangan web, front-end berfokus pada tampilan yang dilihat pengguna, sedangkan back-end mengelola logika program dan pengolahan data pada server.

  • Front-End ➜ UI, tampilan, interaksi pengguna
  • Back-End ➜ database, server, pemrosesan data
  • Keduanya saling terhubung dalam satu sistem
Front-End  ➜ Browser / Client
Back-End   ➜ Server & Database

Pembuatan website dilakukan melalui beberapa tahapan mulai dari perancangan, penulisan kode, pengujian, hingga proses deployment.

  • Analisis & Perancangan
  • Desain Tampilan
  • Pengembangan Front-End & Back-End
  • Testing & Perbaikan Bug
  • Deploy ke Hosting

Desain ➜ Coding ➜ Testing ➜ Deploy ➜ Maintenance

Beberapa tools digunakan untuk membantu proses pengembangan website agar lebih terstruktur dan efisien.

  • VS Code ➜ editor kode
  • Browser (Chrome / Firefox) ➜ menjalankan website
  • Git & GitHub ➜ pengelolaan versi kode
Programmer Web biasanya menggunakan:
VS Code • Terminal • Git • Browser Developer Tools

HTML digunakan untuk menyusun struktur konten halaman web seperti teks, gambar, tautan, tabel, dan elemen lainnya. 


<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertama</title>
  </head>

  <body>
    <h1>Selamat Datang di Pemrograman Web</h1>
    <p>Ini adalah struktur dasar HTML.</p>
  </body>
</html>
