Bab 12

Menambahkan Interaktivitas dengan JavaScript

Pada bab ini kamu akan mempelajari bagaimana JavaScript digunakan untuk menambahkan interaktivitas pada halaman web seperti validasi form, tombol interaktif, serta pembuatan gallery / slider sederhana.

Validasi form memastikan data yang dikirim sesuai aturan, misalnya input tidak boleh kosong atau email harus berformat benar.

Contoh Form


<form id="contactForm">
  <label>Nama</label>
  <input type="text" id="nama" />

  <label>Email</label>
  <input type="email" id="email" />

  <button type="submit">Kirim</button>
</form>

<p id="notif"></p>

Script Validasi


const form = document.getElementById("contactForm");
const notif = document.getElementById("notif");

form.addEventListener("submit", function(e) {
  e.preventDefault();

  const nama = document.getElementById("nama").value;
  const email = document.getElementById("email").value;

  if(nama === "" || email === "") {
    notif.textContent = "Form tidak boleh kosong";
    notif.style.color = "red";
  } else {
    notif.textContent = "Data berhasil dikirim";
    notif.style.color = "green";
  }
});

Tombol interaktif dapat digunakan untuk mengubah tampilan elemen, menampilkan pesan, atau memanipulasi konten halaman. 


<h3 id="textJudul">
  Belajar JavaScript itu Menyenangkan
</h3>

<button onclick="ubahWarna()">
  Ubah Warna Teks
</button>

<script>
function ubahWarna() {
  const judul = document.getElementById("textJudul");
  judul.style.color = "blue";
}
</script>

Gallery atau slider digunakan untuk menampilkan beberapa gambar secara bergantian menggunakan JavaScript. 


<img id="slide" width="320" />

<button onclick="prev()">Prev</button>
<button onclick="next()">Next</button>

<script>
const gambar = [
  "img/slider1.jpg",
  "img/slider2.jpg",
  "img/slider3.jpg"
];

let index = 0;

document.getElementById("slide").src = gambar[index];

function next() {
  index++;
  if(index >= gambar.length) index = 0;
  document.getElementById("slide").src = gambar[index];
}

function prev() {
  index--;
  if(index < 0) index = gambar.length - 1;
  document.getElementById("slide").src = gambar[index];
}
</script>
