JavaScript Modern (ES6+)

Pada bab ini kamu akan mempelajari fitur-fitur modern JavaScript ES6+ yang sering digunakan dalam pengembangan web masa kini seperti let, const, arrow function, template literals, destructuring, dan dasar module import/export.

ES6 memperkenalkan let dan const sebagai pengganti var serta arrow function untuk penulisan fungsi yang lebih ringkas.

  • let ➜ variabel yang masih bisa diubah
  • const ➜ variabel tetap / tidak dapat di-reassign
  • arrow function ➜ bentuk fungsi modern & ringkas

<script>
  let angka = 10;
  const nama = "Hisyam";

  const tambah = (a, b) => a + b;

  console.log(tambah(3, 5));
</script>

Template literals memudahkan pembuatan string dinamis dan destructuring digunakan untuk mengambil nilai dari array atau object secara lebih singkat.

  • Template literals menggunakan backtick (`)
  • Interpolasi variabel dengan ${...}
  • Destructuring object & array

<script>
  const user = { nama: "Ariel", jurusan: "Informatika" };

  const { nama, jurusan } = user;

  const pesan = `Halo ${nama}, jurusan ${jurusan}`;

  console.log(pesan);
</script>

Module digunakan untuk memisahkan kode JavaScript ke dalam beberapa file agar lebih terstruktur dan mudah dipelihara.

  • export ➜ mengirim fungsi / variabel ke file lain
  • import ➜ mengambil module dari file lain
  • digunakan pada aplikasi JavaScript modern

/* file: math.js */
export const tambah = (a, b) => a + b;

/* file: app.js */
import { tambah } from "./math.js";

console.log(tambah(4, 6));
