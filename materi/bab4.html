Bab 4

Pengenalan CSS3

Pada bab ini kamu akan mempelajari dasar penggunaan CSS3 untuk mengatur tampilan halaman web, mulai dari cara menulis CSS, penggunaan selektor dan properti, hingga pengaturan warna, font, dan gaya teks.

CSS dapat diterapkan dengan tiga metode penulisan, yaitu inline langsung pada elemen HTML, internal di dalam tag <style>, serta external melalui file CSS terpisah.

  • Inline CSS ➜ ditulis di atribut style
  • Internal CSS ➜ ditulis di dalam <head>
  • External CSS ➜ file .css terpisah


<p style="color:red;">Contoh teks merah</p>


<style>
  h1 { color: blue; }
</style>


<link rel="stylesheet" href="style.css">

Setiap deklarasi CSS terdiri dari selektor, properti, dan nilai. Selektor menentukan elemen yang akan diberi style, sedangkan properti dan nilai menentukan aturan tampilannya.

  • Selector ➜ elemen yang dipilih
  • Property ➜ aturan yang diatur
  • Value ➜ nilai dari properti

h1 {
  color: green;     /* property */
  font-size: 24px;  /* value */
}

.card {
  padding: 10px;
  border-radius: 8px;
}

CSS menyediakan berbagai properti untuk mengatur warna, jenis huruf, ukuran teks, jarak baris, dan gaya tulisan agar tampilan teks lebih rapi dan mudah dibaca.

  • color ➜ warna teks
  • font-family ➜ jenis huruf
  • font-size ➜ ukuran teks
  • font-weight ➜ ketebalan teks
  • text-align ➜ perataan teks

.title {
  color: #0d6efd;
  font-family: "Segoe UI", Arial;
  font-size: 22px;
  font-weight: 600;
  text-align: center;
}
